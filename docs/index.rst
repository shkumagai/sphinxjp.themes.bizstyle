========================================
 Business style theme sample for Sphinx
========================================

Quick start
===========

install:

.. code-block:: bash

    $ easy_install sphinxjp.themes.rstyle


setup your ``conf.py`` with:

.. code-block:: bash

    extensions = ['sphinxjp.themescore']
    html_theme = 'bizstyle'

and run:

.. code-block:: bash

    $ make html

then your will get this page's style HTML output.
